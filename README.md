# 資格学習アプリ

資格取得を趣味とする人のための、学習を一元管理するフルスタック Web アプリ。
複数の資格を試験日までのカウントダウンとともに管理し、勉強時間の記録、暗記カードによる間隔反復学習、参考書の進捗管理、タスク管理までを 1 つのアプリで完結できる。

## なぜ作ったか

自分自身が資格勉強をする中で、「勉強時間・暗記・進捗の管理がそれぞれ別のツールに散らばっていて不便」と感じたのが出発点。
普段感じている不便を、自分で設計して 1 つのアプリにまとめたいと思い、フロントからバックエンドまで自作した。

> ※ この「なぜ作ったか」は、自分の言葉に書き換えて使ってください。実体験に基づく動機は、作品の一番の強みになります。

## 主な機能

- **資格管理** — 資格の登録・編集・削除。試験日までの残り日数を自動でカウントダウン。
- **勉強時間の記録** — 勉強した時間を記録し、資格ごとの総勉強時間を集計。
- **暗記カード（間隔反復）** — 自作のカードを、忘れた頃に再出題する間隔反復方式で復習。
- **参考書の進捗管理** — 参考書ごとに「今どの章まで進んだか」を記録。
- **タスク管理（Todo）** — 資格ごとにやることリストを管理。

## 技術的なポイント：間隔反復（Leitner システム）

このアプリの中心は、暗記カードの復習スケジューリングを自前で実装した点。
各カードを「箱 1〜5」で管理し、正解すると 1 つ上の箱へ（次の出題が先に延びる）、不正解だと箱 1 に戻す（翌日また出る）。箱の番号が大きいほど出題間隔が長くなる仕組みで、「覚えたカードは間隔を空け、苦手なカードは繰り返す」を自動化している。

| 箱 | 次回出題までの間隔 |
|----|--------------------|
| 1  | 翌日 |
| 2  | 3 日後 |
| 3  | 1 週間後 |
| 4  | 2 週間後 |
| 5  | 1 ヶ月後 |

「今日復習すべきカード」は、各カードの次回出題日が今日以前のものを抽出して表示する。

## 使用技術

**バックエンド**
- Python / FastAPI（REST API。Swagger UI による API ドキュメントを自動生成）
- SQLModel（ORM）
- SQLite（データベース）
- Uvicorn（実行サーバー）

**フロントエンド**
- React（UI）
- Vite（ビルド・開発サーバー）

## 構成

FastAPI で構築した REST API を、React の SPA が呼び出す構成。
データは「資格」を親とし、暗記カード・学習記録・参考書・Todo の 4 種類がひもづくハブ＆スポーク構造になっている。詳しい設計は [`仕様書.md`](./仕様書.md) を参照。

```
React (Vite, :5173)  ──fetch──▶  FastAPI (:8000)  ──▶  SQLite
```

## セットアップと実行

Python 3.10 以上、Node.js 18 以上が必要。バックエンドとフロントエンドを、それぞれ別のターミナルで起動する。

### バックエンド（プロジェクトのルートで）

```bash
python -m venv venv
# 仮想環境を有効化
#   Windows : venv\Scripts\activate
#   Mac/Linux : source venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

起動後、`http://localhost:8000/docs` で API を直接試せる。

### フロントエンド（別ターミナルで）

```bash
cd frontend
npm install
npm run dev
```

`http://localhost:5173` をブラウザで開くとアプリが表示される。
（バックエンドとフロントエンドの両方が起動している必要があります）
