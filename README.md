# 資格学習アプリ

資格取得が趣味の人のための学習管理アプリ。
詳しい設計は [`仕様書.md`](./仕様書.md) を参照。

現在は **スライス 2（学習タイマー）** まで実装済み。

## 必要なもの

- Python 3.10 以上

## セットアップと起動

ターミナル（コマンドプロンプト）でこのフォルダに移動してから、上から順に実行する。

### 1. 仮想環境を作る

プロジェクト専用の隔離された環境を作る（他のプロジェクトに影響しないようにするため）。

```bash
python -m venv venv
```

### 2. 仮想環境を有効化する

Mac / Linux:

```bash
source venv/bin/activate
```

Windows（コマンドプロンプト）:

```bash
venv\Scripts\activate
```

行頭に `(venv)` が表示されれば成功。

### 3. ライブラリをインストールする

```bash
pip install -r requirements.txt
```

### 4. サーバーを起動する

```bash
uvicorn app.main:app --reload
```

`--reload` を付けると、コードを保存するたびに自動で再起動される（開発中に便利）。

### 5. 動作確認する

ブラウザで次を開く：

```
http://127.0.0.1:8000/docs
```

FastAPI が自動生成した API のテスト画面（Swagger UI）が開く。
ここから登録・取得・更新・削除をボタン操作で試せる。**まだ画面（フロント）を作っていなくても、ここで API が動くことを確認できる。**

## 試してみる（/docs での操作例）

1. `POST /certifications` を開き「Try it out」を押す。
2. 下のような JSON を入れて「Execute」：

   ```json
   {
     "name": "基本情報技術者試験",
     "exam_date": "2026-10-18",
     "status": "studying"
   }
   ```

3. レスポンスに `days_until_exam`（試験日までの残り日数）が入って返ってくる。
4. `GET /certifications` で一覧に登録した資格が出れば成功。
